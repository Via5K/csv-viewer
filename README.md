# CSV-Viewer
 * Website designed to view CSV files through Internet.
 * The file is not saved onto the server. It is displayed in form of table "Rows and columns" Which lets visualisation easier.
 
 # View Site

 # Technologies used
 * Jacascript
 * Node JS
 * HTML
 * CSS
